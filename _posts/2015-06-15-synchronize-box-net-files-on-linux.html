I just found this great post on a forum : Linux Mint Forums

Here are the steps to synchronize your box files on Linux with the dolphin file manager :

  1. Open Dolphin, click on network
  2. Click on "Add Network Folder"
  3. Check the box "Webolder (webdav)", click next
  4. Fill in the form :
    1. Name : whatever you want
    2. User : the email address you use to log in box webapps
    3. Server : dav.box.com
    4. Port : 443
    5. Folder : /dav
    6. check BOTH check box "Create an icon..." and "Use encryption"
  5. Click on "Save and Connect"
  6. Enter your password
  7. Optionally right click on the newly created folder and add it to places

Enjoy !
