First install a new desktop environment like xfce :

[code language="bash"]apt-get install kali-defaults kali-root-login desktop-base xfce4 xfce4-places-plugin xfce4-goodies[/code]

Then switch environments with :

[code language="bash"]update-alternatives --config x-session-manager[/code]
